## Key Changes

### Before (Catppuccin)
- Background: Dark Blue (#1E1E2E)
- Text: Light Gray (#CDD6F4)
- Overall: Purple/pink tones

### After (Google CLI / Claude Code Style)
- Background: **Pure Black (#000000)**
- Text: **Dark Blue (#3B82F6)**
- Overall: Clean, professional blue tones
